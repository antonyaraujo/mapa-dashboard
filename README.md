# Mapa Dashboard - Geodatin

Esse projeto, feito em React, conta com a implementação do Google Maps, onde está inserido pontos no mapa filtrados a partir da validação dos dados medidos pela Rede Hidrometeorológica Nacional de Referência do Brasil, o projeto ainda conta com a exibição da localização (longitude e latitude) ao clique do mouse e também com um gráfico de estatísticas que exibe o volume de chuvas ao decorrer dos anos.
